# Wildbeast 🐺

Projeto desenvolvido durante o **curso de CSS Grid Layout da Origamid**.

## 📚 Sobre o Projeto

O objetivo deste projeto foi praticar a criação de layouts utilizando **CSS Grid Layout**, explorando a organização de elementos em linhas e colunas e a construção de páginas responsivas.

Durante o desenvolvimento foram aplicados diversos conceitos importantes do Grid para estruturar a página de forma moderna e flexível.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- CSS Grid Layout
- Google Fonts (Vollkorn)

## 🎯 Conceitos Praticados

- `display: grid`
- `grid-template-columns`
- `grid-template-areas`
- `grid-column` e `grid-row`
- `gap`
- alinhamento com Grid
- layout responsivo com **media queries**

## 📱 Responsividade

O layout se adapta para diferentes tamanhos de tela utilizando **Grid Layout e media queries**, reorganizando as áreas da página para dispositivos menores.

## 🚀 Aprendizados

Durante este projeto pratiquei:

- Construção de layouts complexos com **CSS Grid**
- Organização de estrutura de página utilizando **grid-template-areas**
- Criação de layouts responsivos
- Estruturação de código CSS mais organizada
- Uso de commits seguindo o padrão **Conventional Commits**
